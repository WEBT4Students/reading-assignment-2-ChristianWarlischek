HTML - Basics:

- "!DOCTYPE html" - All HTML documents must start with a document type declaration.


- "h1"-"h6" - HTML headings are defined with the "h1" to "h6" tags.


- "p" - HTML paragraphs are defined with the "p" tag.


- "br" - The "br" tag defines a line break, and is an empty element without a closing tag.


- "hr" - The "hr" tag is an empty tag, which means that it has no end tag.


- "a href" - HTML links are defined with the "a" tag. The link's destination is specified in the "href" attribute.


- "img src","width","height" - HTML images are defined with the "img src" tag. The source file (src), alternative text (alt), width, and height are provided as attributes.


- "style" - The style attribute is used to add styles to an element, such as color, font, size, and more.


- "lang" - You should always include the lang attribute inside the "html" tag, to declare the language of the Web page. This is meant to assist search engines and browsers.


- "title" - The title attribute defines some extra information about an element. The value of the title attribute will be displayed as a tooltip when you mouse over the element.


- "pre" - The HTML "pre" element defines preformatted text.


- "link" - The "link" element definies the relationship between the current document and an external resource. The "link" tag is most often used to link to external style sheets.
